# ✨ Why This Project?

Most resume builders focus only on generating resumes.

This project focuses on:

✅ ATS Optimization

✅ Real-Time Resume Preview

✅ Multiple Premium Templates

✅ Dynamic AI Summary Generation

✅ PDF Export Engine

✅ Responsive Design

✅ Dark / Light Theme

✅ Modern React Architecture

# 🏗 System Architecture

```text
User Input
    ↓
React Components
    ↓
Central Resume State
    ↓
Resume Engine
    ├── ATS Analyzer
    ├── AI Summary Generator
    ├── Template Renderer
    ├── Theme Engine
    └── Preview Engine
    ↓
PDF Export Module
    ↓
Download Resume
```
